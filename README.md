# FiberTech Support

Портал технической поддержки для монтажников связи — подключение интернета, кабель, оптика, диагностика линии и чек-листы выезда.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Деплой

Проект готов для Vercel (Next.js). Репозиторий: [github.com/chubastik/Cursor_test](https://github.com/chubastik/Cursor_test)
